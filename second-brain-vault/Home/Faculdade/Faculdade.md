# Faculdade

Área destinada a todo material acadêmico: disciplinas, conteúdos, provas, exercícios, atividades e projetos.

## Estrutura

```
Faculdade/
└── Semestre-xx/
    └── Nome-da-materia/
        ├── Conteudos/
        ├── Provas-e-exercicios/
        └── Atividades-projetos/
```

## Como usar

1. **Crie uma pasta por semestre:** `Semestre-25-1/`, `Semestre-25-2/`, etc.
2. **Dentro de cada semestre, crie uma pasta por matéria:** `Calculo-1/`, `Fisica-2/`, etc.
3. **Organize o material nas subpastas:**
   - `Conteudos/` — PDFs, slides, lista de vídeos do YouTube relacionados à matéria
   - `Provas-e-exercicios/` — provas anteriores, listas de exercícios resolvidos e não resolvidos
   - `Atividades-projetos/` — atividades e projetos específicos da matéria

## Regras

- Use o formato `Semestre-xx` para manter ordenação cronológica
- Nomeie as matérias com hífens em vez de espaços
- Coloque links para vídeos úteis em `Conteudos/`
- Guarde provas antigas em `Provas-e-exercicios/` para revisão

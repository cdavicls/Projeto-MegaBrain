# Pessoas

Área para contatos, informações e notas sobre pessoas.

## O que colocar aqui

- Contatos (e-mail, telefone, LinkedIn)
- Notas sobre interações
- Perfis completos de cada pessoa
- Histórico de contato

## Como usar

1. **Crie uma pasta por pessoa** com o nome da pessoa: `Nome-Sobrenome/`
2. **Dentro de cada pasta, crie:**
   - `Nome-Sobrenome.md` — perfil principal (use template Person.md)
   - Notas adicionais se necessário

## Regras

- Uma pessoa = uma pasta
- Mantenha o perfil atualizado com informações de contato
- Registre interações importantes na seção "Log" do perfil
- Use o formato `Nome-Sobrenome` com hífens

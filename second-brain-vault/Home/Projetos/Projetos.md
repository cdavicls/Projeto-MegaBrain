# Projetos

Área para projetos pessoais, side projects e ideias que não são de faculdade.

## O que colocar aqui

- Projetos pessoais (apps, sites, ferramentas)
- Side projects e ideias que você quer explorar
- Projetos freelance ou trabalhos extras
- Qualquer iniciativa própria que não seja acadêmica

## Como usar

1. **Crie uma pasta por projeto:** `Nome-do-Projeto/`
2. **Dentro de cada projeto, organize:**
   - `README.md` — descrição, objetivos, status
   - `Tarefas/` — o que precisa ser feito
   - `Notas/` — anotações, decisões, descobertas
   - `Assets/` — arquivos, imagens, documentos

## Regras

- Um projeto = uma pasta
- Mantenha o README atualizado com status
- Arquivos antigos podem ser movidos para `Arquivados/` na raiz de Projetos
